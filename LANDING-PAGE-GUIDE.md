## 🎬 Animation Sequence

### On Page Load (Hero Section)
```
Timeline:
├─ 0.0s: Title fades in + slides up
├─ 0.2s: Subtitle fades in + slides up
├─ 0.4s: Button fades in + slides up
└─ 0.2s: Illustration scales in with bounce
```

### On Scroll (All Sections)
```
Feature Cards:
├─ Card 1: fade in + slide up (on scroll)
├─ Card 2: fade in + slide up (delay +0.15s)
├─ Card 3: fade in + slide up (delay +0.3s)
└─ Card 4: fade in + slide up (delay +0.45s)

Steps:
├─ Step 1: slide left (on scroll)
├─ Step 2: slide right (on scroll)
├─ Step 3: slide left (on scroll)
└─ Step 4: slide right (on scroll)

Benefits:
├─ Benefit 1-4: staggered fade in (on scroll)
```

### On Interaction
```
Button Hover:
├─ Scale: 1 → 1.05
└─ Duration: 0.3s

Card Hover:
├─ Move: y: 0 → -8px
├─ Shadow: light → medium
└─ Duration: 0.4s
```

## 🎯 Key Design Principles Applied

1. **Minimalism**: Only solid colors, no gradients
2. **Hierarchy**: Clear visual hierarchy with size and weight
3. **Spacing**: Consistent padding and margins (1rem = 16px)
4. **Contrast**: Dark text on light, light text on dark
5. **Responsiveness**: Breakpoint at 768px for mobile
6. **Performance**: GPU-accelerated GSAP animations
7. **Accessibility**: Semantic HTML, respects `prefers-reduced-motion`

## 📱 Responsive Behavior

### Desktop (1200px+)
- Hero: 2-column grid (content + illustration)
- Features: 4-column grid
- Steps: 4-column grid
- Benefits: 4-column grid
- Full navbar with all links visible

### Tablet (768px - 1199px)
- Hero: 1-column (stacked)
- Features: 2-column grid
- Steps: 2-column grid
- Benefits: 2-column grid
- Navbar still visible

### Mobile (<768px)
- All sections: 1-column (full stack)
- Hero: Stacked vertically
- Navbar: Minimal (no links shown by default)
- Adjusted font sizes
- Increased padding for touch targets
